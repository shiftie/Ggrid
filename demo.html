Overview


Ggrid is a jQuery plugin that replaces selected pictures by a grid system that fades in cell by cell using CSS3 transitions.

Linkage


Link CSS & JS files to your page : <link rel="stylesheet" href="css/gGrid.css">
<script src="js/jquery.imagesloaded.min.js"></script>
<script src="js/jQuery.gGrid.min.js"></script>

Markup


<img src="bg2.jpg" class="gGrid">

Call


$('img.gGrid').imagesLoaded(function() {
  $('img.gGrid').gGrid();
});

Options


minDelay (default : 0) : minimal delay of the next fadein transition in the row
maxDelay(default : 1.5) : maximal delay of the next fadein transition in the row
cellPerRow (default : 8) : number of cells per row Exemple:
$('img.gGrid').gGrid({minDelay: 0.5, maxDelay: 1.5}); Markup configuration:
<img src="bg.jpg" class="gGrid" data-cell="7">
